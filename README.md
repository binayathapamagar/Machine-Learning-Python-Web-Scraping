# Machine Learning Python Web Scraping Project

This repository contains Python scripts and a Jupyter Notebook that demonstrates web scraping techniques for extracting content from **Medium articles**. The project is divided into two tasks:

## Task 1: Python Script
A Python script that:
- Fetches and extracts the text from a **Medium article**.
- Removes HTML tags.
- Stores the result in a `.txt` file.

## Task 2: Jupyter Notebook
A Jupyter Notebook replicating the functionality of the Python script from **Task 1**, demonstrating:
- **Web scraping** and **file handling** in a notebook format.
- Use of libraries like `requests`, `BeautifulSoup`, and `re` for scraping and processing HTML content.

---

### Key Features:
- **Python** & **Jupyter** for demonstrating web scraping.
- Use of `requests` for HTTP requests.
- `BeautifulSoup` for HTML parsing.
- Regular expressions (`re`) for text manipulation.
  
This project aims to showcase **web scraping skills** in **Python** and **Jupyter**, as well as **version control** and **GitHub collaboration**.

---

### Technologies Used:
- **Python**
- **Jupyter Notebook**
- **requests**
- **BeautifulSoup**
- **re (regular expressions)**

---

### Repository Structure:
- **`task1.py`**: Python script for scraping and saving content.
- **`task1.txt`**: Output text file from Task 1.
- **`task2.ipynb`**: Jupyter Notebook replicating Task 1.
